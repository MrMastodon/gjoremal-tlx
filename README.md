# Gjøremål-TLX

Et enkelt verktøy for å måle opplevd belastning ved hverdagslige gjøremål, basert på NASA Task Load Index (NASA-TLX).
DISCLAIMER: Dette er et vibekodet prosjekt!

## Hva er dette?

NASA-TLX er et veletablert verktøy utviklet av NASA Ames for å måle subjektiv arbeidsbelastning. Denne versjonen er tilpasset for private gjøremål — for å gi deg innsikt i hva som faktisk tapper energi i hverdagen, ikke bare det du tror er slitsomt.

## Slik bruker du det

1. Skriv inn navnet på gjøremålet du skal score
2. Ranger seks belastningsdimensjoner på en skala fra 0 til 100
3. Gjennomfør 15 parvise sammenligninger for å angi hva som veide tyngst
4. Resultatet beregnes automatisk og lagres

## De seks dimensjonene

| Dimensjon | Beskrivelse |
|---|---|
| 🧠 Mentalt krav | Konsentrasjon og mental kompleksitet |
| 💪 Fysisk krav | Fysisk aktivitet og anstrengelse |
| ⏱️ Tidspress | Opplevd hastverk og tempo |
| 🎯 Resultat | Hvor fornøyd du er med egen innsats |
| ⚡ Innsats | Total innsats, mentalt og fysisk |
| 😤 Frustrasjon | Irritasjon, stress og frustrasjon |

## Datalagring

All data lagres lokalt i nettleseren din via `localStorage` — ingenting sendes til noen server. Du kan eksportere og importere data som JSON-fil under innstillinger (⚙).
